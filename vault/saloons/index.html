The Refreshment Saloon Record

Follow a soldier, a unit, or a volunteer through the Cooper Shop and Union Volunteer Refreshment Saloon record — 1861 to 1865.

Prototype notice — this explorer is connected to the live Airtable research base and loads the full dataset: ~1,478 Cooper Shop Ledger entries, ~1,422 UVRS diary entries, ~1,089 units, and ~112 leadership records. Data loads on page open; run npm run sync-airtable from the project root to refresh the local JSON cache.
